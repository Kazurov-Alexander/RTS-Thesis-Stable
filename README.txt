RTS Thesis Project
==================

Описание:
----------
Это дипломный проект, реализующий базовую механику стратегии в реальном времени (RTS)
с использованием библиотеки PyGame. Игра вдохновлена RimWorld и включает карту, юнитов,
здания, действия и игровой цикл.

Функциональность:
------------------
- Отображение игрового окна
- Клеточная карта (по идее гексы)
- Перемещение юнита
- Игровой цикл с FPS
- Структурированный код и ресурсы

Структура проекта:
-------------------
assets/         - ресурсы проекта
│
├── images/         - все изображения
│   ├── tiles/          - тайлы карты: земля, вода, трава и т.д.
│   ├── units/          - спрайты юнитов
│   │   ├── player/         - изображения игрока
│   │   ├── enemy/          - изображения врагов
│   │   └── neutral/        - нейтральные персонажи
│   ├── buildings/      - изображения построек
│   │   ├── storage/        - склады, хранилища
│   │   ├── defense/        - башни, стены
│   │   └── production/     - мастерские, генераторы
│   ├── ui/             - элементы интерфейса: кнопки, иконки
│   └── effects/        - визуальные эффекты: огонь, дым, взрывы
│
├── sounds/         - звуковые эффекты
│   ├── ambient/        - атмосферные звуки: ветер, дождь
│   ├── actions/        - действия: шаги, атаки, клики
│   └── ui/             - звуки интерфейса: нажатия, уведомления
│
├── music/          - фоновая музыка
├── fonts/          - шрифты для текста и интерфейса
├── animations/     - анимации
│   ├── units/          - движения, атаки юнитов
│   └── buildings/      - постройка, разрушение зданий
├── data/           - конфигурационные файлы
│   ├── maps/           - пресеты карт
│   ├── factions/       - параметры фракций
│   └── balance/        - баланс: урон, скорость, стоимость
└── temp/           - временные ресурсы, черновики

src/             - исходный код
├── main.py          - точка входа
├── settings.py      - настройки экрана, цветов, FPS
├── game.py          - игровой цикл
├── map.py           - генерация карты
├── player.py        - логика юнита
├── ...              - дополнительные модули

requirements.txt - зависимости (pygame и др.)
README.txt- описание проекта

Установка:
----------
1. Установить Python 3.10+
2. Создать виртуальное окружение:
   python -m venv .venv
3. Активировать окружение:
   - Windows: .venv\Scripts\activate
   - Linux/macOS: source .venv/bin/activate
4. Установить зависимости:
   pip install -r requirements.txt

Запуск:
-------
python src/main.py
